Selector Description
"*" Universal selector, matches any element.
".classname" Class selector, matches all elements with class="classname".
"#id" ID selector, matches an element with id="id".
"element" Type selector, matches elements of a specific type.
"element,element" Grouping selector, selects all matching elements.
"element element" Descendant selector, matches elements inside other elements.
"element>element" Child selector, matches direct children of an element.
"element+element" Adjacent sibling selector, matches a direct next sibling.
"[attribute]" Attribute selector, matches elements with a specific attribute.
"[attribute=value]" Attribute equals selector, matches elements with attribute="value".
":hover" Pseudo-class, applies when the user hovers over an element.
":focus" Pseudo-class, applies when the element has focus.
":nth-child(n)" Pseudo-class, matches the nth child of its parent.
